Portable Data Loggers

Professional Portable Data Acquisition for Temperature & Humidity Monitoring

Product Description

Portable Data loggers are used in various industries to display the temperature & humidity and store these values for further use.

These Portable Data Loggers monitor, display (via a 4 Digit LCD), and store temperature and humidity (2 Channels) using a Digital Sensor and Non-volatile memory. Key features include high accuracy, durability, mobility, an Inbuilt RTC, PC software, and power via an inbuilt rechargeable battery. Data can be transferred via RS 232 or USB, and optional features include a buzzer and programmable alarm set points.
